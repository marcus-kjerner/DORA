DORA

Device Operations and Registration Application

Vad är DORA?
DORA är ett internt verktyg för IT-avdelningen i Söderköpings kommun. Appen förenklar inventariehanteringen genom att låta personal utföra vanliga administrativa åtgärder direkt på plats via mobilen. Genom att scanna en enhets stöld-ID kopplar appen upp sig mot Easit GO och tillåter användaren att skrota utrustning, flytta enheter till IT-lager eller registrera ett ägarbyte med bara några tryck.

Hur appen används

  1. Logga in: Öppna appen och identifiera dig med FaceID eller fingeravtryck.
  2. Välj åtgärd: Välj om du vill Skrota, flytta till IT-Lager eller Flytta inventarie (ägarbyte).
  3. Scanna: Rikta kameran mot enhetens stöld-ID.
  4. Bekräfta: Kontrollera att informationen som visas (Modell, nuvarande Ägare etc.) stämmer.
  5. Utför: Vid flytt väljer du ny användare och byggnad/rum i rullistorna. Tryck sedan på bekräfta för att uppdatera Easit i realtid.

Vanliga frågor (FAQ)

Vad gör jag om en streckkod är skadad och inte kan scannas?

Om kameran inte kan läsa av streckkoden kan du klicka på tangentbordsikonen i det övre högra hörnet på scannerskärmen. Där kan du skriva in enhetens stöld-ID (6 siffror) manuellt för att hämta informationen från Easit.

Varför kräver appen fingeravtryck eller ansiktsigenkänning vid start?

För att skydda kommunens data och säkerställa att din personliga API-nyckel inte missbrukas kräver DORA biometrisk autentisering. Detta säkerställer att endast behörig personal kan utföra ändringar i inventariesystemet om telefonen skulle hamna i orätta händer.

Hur vet jag att en flytt eller skrotning faktiskt har genomförts i Easit?

När du bekräftar en åtgärd skickar appen ett direktanrop till Easit. Om allt går bra visas ett grönt meddelande: "✔ Uppdatering lyckades!". Dessutom läggs enheten till i historiklistan längst ner på skärmen under din nuvarande session, så att du enkelt kan dubbelkolla vad du nyss har gjort.

Krävs VPN för att använda appen utanför kommunhuset?

Ja, eftersom appen kommunicerar direkt med kommunens interna integrations-API för Easit, måste din tjänstetelefon ha en aktiv anslutning till kommunens nätverk (antingen via WiFi på plats eller via Always-on VPN) för att kunna hämta och skicka data.

Hur hanterar jag min kontodata och åtkomst?

Eftersom DORA är en intern företagsapp knuten till Söderköpings kommuns system, hanteras din behörighet via kommunens IT-avdelning. Om du vill återkalla din åtkomst, ta bort din personliga API-nyckel eller avsluta ditt konto, vänligen kontakta den interna IT-supporten direkt eller via formuläret nedan.
